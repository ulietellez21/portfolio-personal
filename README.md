# Portfolio Personal — Ulises Téllez Landón

Portfolio personal desarrollado con React, TypeScript, Tailwind CSS y Framer Motion.

## Tech Stack

- **React 18** + **TypeScript**
- **Tailwind CSS v3** — utilidades de estilos
- **Framer Motion** — animaciones fluidas
- **Vite** — bundler y dev server
- **Lucide React** — iconografía

## Características

- Diseño responsivo (mobile-first)
- Modo oscuro/claro con toggle
- Animaciones de entrada al hacer scroll
- Secciones: Hero, Sobre mí, Skills, Proyectos, Contacto
- Listo para deploy en GitHub Pages

## Instalación

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
